# ScrollView


距离底部100dp，无效

<ScrollView xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:ignore="HardcodedText">

    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_marginBottom="100dp"
        android:orientation="vertical">

        <TextView ... />
    </LinearLayout>
</ScrollView>



在内部元素的最后一个容器配置外边距，有效，但有时元素是动态增减的，适用性较差。

<ScrollView xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:ignore="HardcodedText">

    <!--        android:padding="10dp"-->
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_marginBottom="100dp"
        android:orientation="vertical"
>

        <TextView
            android:margin="10dp" />
    </LinearLayout>
</ScrollView>


对内部容器配置padding，有效，因为此时距离在内部容器与其内容之间，而不是ScrollView与容器之间

<ScrollView xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:ignore="HardcodedText">

    <LinearLayout
    android:padding="10dp"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_marginBottom="100dp"
        android:orientation="vertical">

        <TextView ... />
    </LinearLayout>
</ScrollView>


# Glide

//Glide设置图片圆角角度
    RoundedCorners roundedCorners = new RoundedCorners(10);
    //通过RequestOptions扩展功能,override:采样率,因为ImageView就这么大,可以压缩图片,降低内存消耗
    // RequestOptions options = RequestOptions.bitmapTransform(roundedCorners).override(20, 20);
    RequestOptions options = RequestOptions.bitmapTransform(roundedCorners);
    Glide.with(context)
            .load(lc.getVod_pic().toString())
            //.placeholder(R.drawable.ic_default_image)
            .error(R.drawable.ic_clipimage)
            .apply(options)
