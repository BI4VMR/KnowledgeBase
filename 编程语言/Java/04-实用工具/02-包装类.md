# 概述
基本数据类型不具备对象的特性，功能简单，为了让基本数据类型也具备对象的特性，Java为每个基本数据类型提供了一个包装类，这样我们就可以像操作对象一样来操作它们了。

基本数据类型与包装类之间的对应关系如下表所示：

<div align="center">

| 数据类型 |  包装类   |
| :------: | :-------: |
|   byte   |   Byte    |
|  short   |   Short   |
|   int    |  Integer  |
|   long   |   Long    |
|  float   |   Float   |
|  double  |  Double   |
|   char   | Character |
| boolean  |  Boolean  |

</div>

包装类主要提供了将本类型和其他基本类型互相转换的方法。此处以Integer类为例：

🔷 `String toString()`
<br />
将数值转换为String类型。

🔷 `int parseInt(String s)`
<br />
将输入值转为int类型。

🔷 `Integer valueOf(String s)`
<br />
将输入值转为Integer类型。

# 自动装箱/拆箱
基本类型和包装类之间经常需要互相转换。把基本类型转换成包装类，使其具有对象的性质，就是装箱。和装箱相反，把包装类对象转换成基本类型的值，则是拆箱。

早期装箱与拆箱都需要手动操作，使用 `valueOf()` 方法将输入转换为包装类，再转为基本类型。从JDK 1.5开始，Java引入了自动装箱和拆箱机制，包装类和基本类型之间的转换就更加轻松便利了，我们可以使用包装类的 `parseXXX()` 系列方法直接获取转换后的基本类型数据，且这些方法是静态的，可以直接使用类名调用。

```java
// 定义double类型变量
double i = 91.5;
// 手动装箱
Double dm = new Double(i);
// 自动装箱
Double da = i;
System.out.println("装箱后的结果：" + dm + "和" + da);

// 定义一个Float包装类对象，值为87.3
Float j = new Float(87.3);
// 手动拆箱
float k = j.floatValue();
// 自动拆箱
float l = j;
System.out.println("拆箱后的结果为：" + k + "和" + l);

// 将字符串直接转为int型(使用静态方法)
int t = Integer.parseInt("101");
System.out.println("转换后的结果为：" + t);
```

运行程序并查看控制台输出：

```text
装箱后的结果：91.5和91.5
拆箱后的结果为：87.3和87.3
转换后的结果为：101
```
