# 简介
多态(Polymorphism)是面向对象编程的三大特性之一。当我们通过父类（或接口）类型的变量调用方法时，系统会根据变量实际指向的对象类型，自动决定执行哪个类的实现，这就是"方法多态"。

多态是框架设计的基础，它允许我们以统一的方式操作不同类型的对象。

# 基本应用
🔴 示例一：多态的基本应用。

在本示例中，我们声明Father类型的变量，分别指向Father和Child的实例，并调用同一方法，观察不同的输出。

"Father.kt":

```kotlin
open class Father {
    open fun show() {
        println("This is Father.")
    }
}
```

"Child.kt":

```kotlin
class Child : Father() {
    override fun show() {
        println("This is Child.")
    }
}
```

"TestPolymorphism.kt":

```kotlin
// 声明Father变量"t1"，指向父类的引用。
val t1: Father = Father()
// 声明Father变量"t2"，指向子类的引用。
val t2: Father = Child()

// 分别执行两个变量的"show()"方法
t1.show()
t2.show()
```

此时运行示例程序，并查看控制台输出信息：

```text
This is Father.
This is Child.
```

# 类型判断与智能转换
## "is"关键字
Kotlin使用"is"关键字判断对象的类型，对应Java的"instanceof"。Kotlin还提供了"智能转型(Smart Cast)"：经过"is"判断后，在对应的代码块中该变量会被编译器**自动转换**为对应的类型，无需手动强制转换。

🟠 示例二：类型判断与智能转型。

"TestPolymorphism.kt":

```kotlin
open class Animal
class Dog : Animal() {
    fun bark() = println("汪汪！")
}
class Cat : Animal() {
    fun meow() = println("喵喵！")
}

val animals: List<Animal> = listOf(Dog(), Cat(), Dog())

for (animal in animals) {
    if (animal is Dog) {
        // 智能转型：此处animal已自动转为Dog类型
        animal.bark()
    } else if (animal is Cat) {
        animal.meow()
    }
}
```

此时运行示例程序，并查看控制台输出信息：

```text
汪汪！
喵喵！
汪汪！
```

## 类型转换
使用"as"关键字可以显式地将对象转换为指定类型；若转换失败会抛出 `ClassCastException` 。

使用"as?"进行安全转换：若转换失败则返回 `null` ，不会抛出异常。

🟡 示例三：显式类型转换。

"TestPolymorphism.kt":

```kotlin
val obj: Any = "Hello, Kotlin!"

// 使用as进行强制类型转换
val str1: String = obj as String
println("强制转换结果：$str1")

// 使用as?进行安全转换（失败时返回null）
val num: Any = 42
val str2: String? = num as? String
println("安全转换结果：$str2")
```

此时运行示例程序，并查看控制台输出信息：

```text
强制转换结果：Hello, Kotlin!
安全转换结果：null
```
