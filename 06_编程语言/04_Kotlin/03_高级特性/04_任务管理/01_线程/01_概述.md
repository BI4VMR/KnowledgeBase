# 实用工具
Kotlin提供了便捷方法用于创建并配置线程，我们可以按需使用。

```kotlin
public fun thread(
    start: Boolean = true,
    isDaemon: Boolean = false,
    contextClassLoader: ClassLoader? = null,
    name: String? = null,
    priority: Int = -1,
    block: () -> Unit
): Thread {
    // 此处已省略部分代码...
}
```

下文列表将对该方法的参数进行说明：

- `start` : 是否立刻启动线程。如果我们将该变量置为 `false` ，后续可以手动调用 `Thread` 实例的 `start()` 方法开启线程。
- `isDaemon` : 是否为守护线程
- `contextClassLoader` : 指定ClassLoader
- `name` : 设置线程名称。
- `priority` : 设置优先级
- `block` : 在线程 `run()` 方法中执行的语句。
