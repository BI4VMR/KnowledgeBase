# 分支结构
## 简介
有时我们需要根据特定的条件，执行不同的动作，此时可以使用分支结构组织代码。

## "if"语句
"if"语句用于"仅当满足条件后才执行动作"的情况，该语句的结构如下文代码块所示：

```text
// 满足条件时执行多条语句
if (CONDITION) {
    STATEMENT
}
OTHER

// 满足条件时执行单条语句
if (CONDITION) STATEMENT
OTHER
```

条件"CONDITION"可以是一个布尔型变量，也可以是结果为布尔值的表达式。当表达式的值为"true"时，程序将会首先执行"STATEMENT"所指代的语句，然后再执行后续语句"OTHER"；当表达式的值为"false"时，则跳过"STATEMENT"所指代的语句，直接执行后续语句"OTHER"。

## "if-else"语句
"if-else"语句相比"if"语句多了一个分支，当条件成立时，执行"if"分支内部的代码；条件不成立时，则执行"else"分支内部的代码。

"if-else"语句的结构如下文代码块所示：

```text
if (CONDITION) {
    STATEMENT I
} else {
    STATEMENT II
}
```

当条件"CONDITION"为真时，执行语句"STATEMENT I"；条件为假时，则执行语句"STATEMENT II"。

## 多重"if-else"语句
多重"if-else"语句通常用于判断数值的所属区间，它的语法如下文代码块所示：

```text
if (CONDITION I) {
    STATEMENT I
} else if (CONDITION II) {
    STATEMENT II
} else if (CONDITION III) {
    STATEMENT III
} else {
    STATEMENT E
}
OTHER
```

当条件"CONDITION I"不满足的情况下，才会进行条件"CONDITION II"的判断，以此类推；如果某个条件成立，则执行对应的"STATEMENT"语句，然后结束判断转至"OTHER"执行后续语句；如果所有条件均不成立，才会执行"else"块内的语句。

## "when"语句
Kotlin中没有"switch"语句，取而代之的是功能更强大的"when"语句。"when"语句将输入值与多个分支逐一匹配，它的语法如下文代码块所示：

```text
when (INPUT) {
    VALUE I -> STATEMENT I
    VALUE II -> STATEMENT II
    else -> STATEMENT E
}
OTHER
```

下文列表对"when"语句的逻辑与特性进行了详细地说明：

- 输入值"INPUT"可以是任意类型，包括整型、字符串、枚举等。
- 每个分支由匹配值"VALUE"、箭头符号" `->` "和对应的语句"STATEMENT"组成，无需"break"语句，分支匹配成功后不会继续匹配其余分支。
- 如果某个分支需要执行多条语句，可以用大括号将它们包裹起来。
- 使用逗号( `,` )可以将多个匹配值合并为一个分支，实现相同的处理逻辑。
- 使用 `in` 关键字可以匹配某个区间或集合，例如 `in 1..3` 表示匹配1到3之间的整数。
- "else"分支与Java"switch"中的"default"语义相同，在所有分支均未匹配时执行；当所有可能的输入值都已被其他分支覆盖时，"else"分支可省略。

## 分支结构的嵌套
条件语句内部可以放置任意其他语句，包括另一个条件语句，我们可以通过嵌套条件语句实现较为复杂的逻辑。

🟤 示例七：判断输入值与区间的关系。

在本示例中，我们给定一个数字，判断它与区间 `(4, 6)` 的关系。

"TestBranch.kt":

```kotlin
val x = 5
if (x > 4) {
    if (x > 6) {
        println("x大于6")
    } else {
        println("x在4到6之间")
    }
} else {
    println("x小于等于4")
}
```

此时运行示例程序，并查看控制台输出信息：

```text
x在4到6之间
```

> 🚩 提示
>
> 本示例仅用于演示分支结构的嵌套用法，我们更推荐使用多重"if-else"语句实现区间判断，因为它的缩进层次更少，可读性更高。

# 循环结构
## 简介
为了完成某些任务，我们需要重复执行一系列动作，直到满足条件为止，此时可以使用循环结构组织代码。

## "while"语句
"while"语句是一种简单的循环结构，它的语法如下文代码块所示：

```text
while (CONDITION) {
    STATEMENT
}
OTHER
```

当条件"CONDITION"成立时，执行循环体"STATEMENT"，然后再次判断条件是否成立，若成立则继续执行循环体；如此往复，直到条件不成立为止，退出循环并执行OTHER所指代的后续语句。

🔴 示例八："while"语句的基本应用。

在本示例中，我们向控制台输出从"1"到"5"的所有整数。

"TestLoop.kt":

```kotlin
var i = 1
// 当变量"小于等于5"时执行循环
while (i <= 5) {
    // 输出变量的值，并且使变量加1。
    println(i)
    i++
}
```

在上述代码中，每执行一次循环体，变量 `i` 的值都会增加"1"；当输出"5"之后， `i` 自增为"6"，不再满足循环条件 `i <= 5` ，因此退出循环，程序终止。

```text

2
3
4
5
```

## "do-while"语句
"do-while"语句与"while"语句类似，不同之处在于"while"语句先进行条件判断，再决定是否执行循环体；而"do-while"语句会无条件执行一次循环体，再进行条件判断，确定是否仍要执行循环体。

"do-while"语句的语法如下文代码块所示：

```text
do {
    STATEMENT
} while (CONDITION)
```

## "for"语句
Kotlin的"for"语句通常与区间(Range)结合使用，可以方便地遍历一个连续的数值范围，它的语法如下文代码块所示：

```text
for (<循环变量> in <区间或集合>) {
    <STATEMENT>
}
```

常用的区间表达式如下文列表所示：

- `1..5` ：闭区间，包含1、2、3、4、5。
- `1 until 5` ：左闭右开区间，包含1、2、3、4，不含5。
- `5 downTo 1` ：降序闭区间，包含5、4、3、2、1。
- `1..10 step 2` ：带步长的区间，包含1、3、5、7、9。

## 循环结构的嵌套
如果一个循环结构的循环体中包含另一个循环结构，则被称为"多重循环"，最常见的就是二重"for"循环结构。在二重循环中，外层循环每执行一次，内层循环都要完整地执行一遍。

🟢 示例十一：在控制台上输出三角形。

在本示例中，我们使用"*"在控制台上输出直角三角形。

"TestLoop.kt":

```kotlin
// 外层循环控制行数
for (i in 1..5) {
    // 内层循环控制每行的*号数
    for (j in 1..i) {
        print("*")
    }
    // 每打印完一行后进行换行
    println()
}
```

此时运行示例程序，并查看控制台输出信息：

```text
*
**
***
****
*****
```

## 循环控制语句
### "break"语句
有时我们不知道循环体需要被执行的总轮次，但知道终止条件，此时可以将循环轮次设为无限，并在循环体内部判断是否满足终止条件，当条件满足时，使用"break"语句终止循环。

🔵 示例十二："break"语句的基本应用。

在本示例中，我们累加从"1"开始的正整数，并找出使和超过"20"的数字。

"TestLoop.kt":

```kotlin
// 保存累加数值
var sum = 0
var i = 1
while (true) {
    // 每次循环时累加求和
    sum += i
    // 判断累加数值是否大于20，如果满足条件则直接退出循环
    if (sum > 20) {
        println("当前的整数为: $i")
        println("当前的累加数值为: $sum")
        // 退出循环
        break
    }
    i++
}
```

在上述代码中，"while(true)"将会无限循环。每轮循环执行时，都会经过一次"if"语句，一旦条件满足，就会执行"break"语句，随后整个循环被终止。

此时运行示例程序，并查看控制台输出信息：

```text
当前的整数为: 6
当前的累加数值为: 21
```

### "continue"语句
"continue"语句的作用是跳过当前轮次循环体中的剩余语句，直接进行下一轮循环。

🟣 示例十三："continue"语句的基本应用。

在本示例中，我们计算"1"到"10"之间所有偶数之和。

"TestLoop.kt":

```kotlin
// 保存累加值
var sum = 0
for (i in 1..10) {
    // 如果i为奇数，不执行累加操作，直接结束本轮循环，进入下一轮循环操作。
    if (i % 2 != 0) {
        continue
    }
    sum += i
}

print("1到10之间的所有偶数的和为：$sum")
```

此时运行示例程序，并查看控制台输出信息：

```text
1到10之间的所有偶数的和为：30
```

### 标签跳转
Kotlin支持标签(Label)语法，可以在"break"或"continue"语句中指定要跳出或继续的具体循环层级，这在多重循环中非常有用。

标签使用"标签名@"的形式定义在循环语句之前，跳转时使用 `break@标签名` 或 `continue@标签名` 语法。

🟤 示例十四：使用标签跳出多重循环。

在本示例中，我们在二重循环中找到目标元素后，立即跳出外层循环。

"TestLoop.kt":

```kotlin
outer@ for (i in 1..3) {
    for (j in 1..3) {
        if (i == 2 && j == 2) {
            println("找到目标：i=$i, j=$j，退出所有循环。")
            break@outer
        }
        println("i=$i, j=$j")
    }
}
```

此时运行示例程序，并查看控制台输出信息：

```text
i=1, j=1
i=1, j=2
i=1, j=3
i=2, j=1
找到目标：i=2, j=2，退出所有循环。
```
