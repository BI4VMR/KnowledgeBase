# 一维数组
## 简介
数组是一种数据结构，可以按顺序存储多个类型相同的数据值。数组中的元素通过"下标"来访问，下标是数组中元素位置的索引，下标从0开始，0表示第一个元素，1表示第二个元素，以此类推，直到"数组长度-1"。如果我们试图访问一个不存在的下标，程序就会出错。

## 基本应用
Kotlin提供了两类数组：

- **基本类型数组**：`IntArray`、`LongArray`、`FloatArray`、`DoubleArray`、`BooleanArray`、`CharArray`等，底层存储基本类型，性能更高。
- **泛型数组**：`Array<T>`，可存储任意对象类型，例如 `Array<String>`。

常用的创建数组方式如下文代码块所示：

```text
// 使用工厂方法创建（推荐）
val array = intArrayOf(1, 2, 3)
val strArray = arrayOf("A", "B", "C")

// 创建指定长度的数组（元素为默认值）
val array2 = IntArray(5)

// 创建指定长度的数组，并通过Lambda初始化每个元素
val array3 = IntArray(5) { it * 2 }  // [0, 2, 4, 6, 8]

// 给下标对应的元素赋值
array[0] = 10
// 访问下标对应的元素
val first = array[0]
```

下文代码中我们声明了一个字符串数组，并通过下标访问第四个元素。

```kotlin
// 定义字符串数组，保存考试科目信息
val subjects = arrayOf("Oracle", "PHP", "Linux", "Kotlin", "HTML")

// 访问第四个元素（下标从0开始）
println("数组中第4个科目为：${subjects[3]}")
```

运行程序并查看控制台输出：

```text
数组中第4个科目为：Kotlin
```

数组的长度一旦确定，后续就不能再更改了。通过 `.size` 属性可以获取数组的长度。如果我们想要使用长度可以自由伸缩的容器，可以使用List等集合。

## 初始化数组
创建数组后，各元素的初始值为当前数据类型的默认值，我们有时可能需要将所有元素都设为某个特定值，此时可以使用 `fill()` 方法快速完成初始化。

`fill(element)` 可以更改所有元素； `fill(element, fromIndex, toIndex)` 方法可以更改指定区间的元素，左闭右开。

```kotlin
// 创建包含三个元素的整型数组
val array = IntArray(3)
// 查看元素的默认值
print("默认值：")
print("${array[0]} ")
print("${array[1]} ")
println(array[2])

// 初始化数组，将所有元素的值设为100。
array.fill(100)
// 查看填充后各元素的值
print("填充后：")
print("${array[0]} ")
print("${array[1]} ")
print(array[2])
```

运行程序并查看控制台输出：

```text
默认值：0 0 0
填充后：100 100 100
```

## 遍历数组
### 使用循环语句操作数组
我们经常使用"for"循环操作数组，访问元素或者给元素赋值。通过数组的 `.size` 属性，我们可以获取数组的长度，确定循环结束条件；也可以使用 `indices` 属性遍历所有下标。

```kotlin
// 定义一个字符串数组。
val hobbys = arrayOf("sports", "game", "movie")
println("循环输出数组中元素的值：")
// 使用for循环遍历数组中的元素
for (i in hobbys.indices) {
    println(hobbys[i])
}
```

运行程序并查看控制台输出：

```text
循环输出数组中元素的值：
sports
game
movie
```

### 使用"for-in"操作数组
当我们想要访问数组中的每一个元素时，可以使用"for-in"循环直接遍历元素，比使用下标更加简洁。

```kotlin
// 定义一个整型数组，保存成绩信息
val scores = intArrayOf(89, 72, 64, 58, 93)
// 使用for-in遍历输出数组中的元素
for (score in scores) {
    print("$score; ")
}
```

运行程序并查看控制台输出：

```text
89; 72; 64; 58; 93;
```

## 复制数组
数组本质上是一种对象，数组变量保存的是元素在堆内存中的地址。由此可知，当我们需要复制数组内容时，不能直接将原数组赋值给新的变量。

```kotlin
val array1 = intArrayOf(1, 2, 3)
// 定义"array2"并将"array1"赋值给它（引用复制，非内容复制）
val array2 = array1
// 更改"array1"的第一个元素
array1[0] = 10
// 输出"array2"的第一个元素，会随"array1"一起变化
println(array2[0])
```

运行程序并查看控制台输出：

```text
10
```

当我们需要复制数组时，应当使用 `copyOf()` 或 `copyOfRange(from, to)` 方法。

```kotlin
// 创建包含3个元素的整型数组
val array1 = intArrayOf(1, 2, 3)
// 复制全部数组
val array2 = array1.copyOf()
// 修改array1不影响array2
array1[0] = 10
println("array1[0] = ${array1[0]}")
println("array2[0] = ${array2[0]}")
```

运行程序并查看控制台输出：

```text
array1[0] = 10
array2[0] = 1
```

## 使用扩展方法操作数组
Kotlin为数组提供了丰富的扩展方法，可以实现排序、打印等功能。

常用的扩展方法如下文内容所示：

🔷 `sort()`

此方法可以实现对元素的排序，按升序排列。

🔷 `contentToString()`

此方法可以将数组转换成字符串，输出格式为：每个元素顺次连接，元素之间使用逗号和空格隔开。

🔴 示例一

尝试使用上述方法操作数组。

```kotlin
val scores = intArrayOf(89, 72, 64, 58, 93)
// 使用"sort()"方法对数组进行排序
scores.sort()

print("升序排列结果：")
for (score in scores) {
    print("$score;")
}
println()

// 使用"contentToString()"方法将数组转换为字符串
val str = scores.contentToString()
print("数组内容：$str")
```

运行程序并查看控制台输出：

```text
升序排列结果：58;64;72;89;93;
数组内容：[58, 64, 72, 89, 93]
```

# 二维数组
## 简介
多维数组就是"数组的元素也是一个数组"，此处以二维数组为例进行说明。

二维数组可以简单地理解为一种"特殊"的一维数组，它的每个元素都嵌套了一个完整的一维数组。

二维数组的声明语法如下文所示：

```kotlin
// 使用arrayOf + intArrayOf声明2行3列的二维整型数组
val matrix = arrayOf(
    intArrayOf(1, 2, 3),
    intArrayOf(4, 5, 6)
)

// 通用泛型写法
val names: Array<Array<String>> = arrayOf(
    arrayOf("tom", "jack", "mike"),
    arrayOf("张三", "李四", "王五")
)
```

🔴 示例二

尝试声明并访问二维数组。

```kotlin
// 定义2行3列的二维数组并赋值
val names = arrayOf(
    arrayOf("tom", "jack", "mike"),
    arrayOf("张三", "李四", "王五")
)
// 通过二重循环输出二维数组中元素的值
for (row in names) {
    for (element in row) {
        println(element)
    }
    println()
}
```

运行程序并查看控制台输出：

```text
tom
jack
mike

张三
李四
王五
```

上述代码中，我们使用二重"for-in"循环结构访问二维数组，其中外层循环遍历每一行，内层循环遍历每行中的元素。
